GitHub Get Started

Swell Kernel

Monolithic, performance-patched, no modules. All common x86_64 drivers baked in. Every config option documented.

runit

Simple, fast init. No systemd. No bloat. Base services only — add what you need.

sbpm

Swell Blazing Package Manager. Source-based. Written in Rust. Simple topological dependency resolution.

KDE Plasma

Modern desktop on Wayland. PipeWire audio. Fully opt-in — base is CLI only.

Ext4

Bulletproof filesystem. Btrfs available at install time. Simple, reliable, boring.

Rolling + Freeze

Rolling release with sbpm freeze to pin everything. Update on your terms.

Install

Boot the live ISO and run:

swell-install

Follow the TUI — partition, select metapackages, build from source, reboot.

Package Manager

sbpm -S firefox          # install
sbpm -R firefox          # remove
sbpm -U                  # upgrade all
sbpm -Ss multimedia      # search
sbpm -Syu                # sync + upgrade
sbpm freeze              # pin all packages
